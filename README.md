# Kali_Rice
### Bash script to automate the install of some custom dot files and othe repos on Kali Linux
---
##### Quick install:
`git clone https://github.com/LinuxUser255/Kali_Rice.git`

 ```
 cd Kali_Rice
 pip install -r requirements.txt
 chmod +x rice.sh
 ./rice.sh
 ```
also run: 
```
./install-coc-nvim.sh
```
